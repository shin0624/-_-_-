# 차별점
- Reality 
    1. 실제 한국 대형병원의 특징을 살린 그래픽을 사용하여 플레이어에게 현실감 제공
    2. 익숙한 배경에서 느끼는 공포와 스릴 -> 높은 게임성 체감 가능
