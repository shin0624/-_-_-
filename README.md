# 개요
    OUTLAST, Resident Evil, Silint Hill 등 공포 장르 스테디셀러는 모두 해외 배경의 게임입니다.
    한국을 배경으로 한 공포게임이라면 한국 유저의 몰입도를 더욱 높일 수 있다는 의견을 시작으로, HERESIS가 기획되었습니다.
